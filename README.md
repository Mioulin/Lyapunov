# Lyapunov: dynamics of uncertainty in high-dimensional stochastic systems

> *From brain trajectories to LLM reasoning paths: a single number that tells you whether your system is anchored or about to slip.*

## What this is

A small, framework-agnostic Python toolkit for estimating the **largest Lyapunov exponent (λ)** of trajectories in a reconstructed phase space.

It started life as a pipeline for **6-dimensional brain trajectories** (3 functional-connectivity gradients + their first derivatives, n=100 Schaefer parcels) under psychedelic and resting-state conditions. The same code, with no changes, estimates λ on any time series where you can build or reconstruct a phase space.

This README explains *why* λ is worth caring about - especially for **high-dimensional stochastic data** and **modern LLM evaluation**.

---

## The 60-second version

The Lyapunov exponent λ answers one question: *how fast do nearby trajectories diverge?*

| Sign | Geometry | Meaning |
|---|---|---|
| **λ < 0** | nearby paths collapse | stable, predictable, attractor pulls trajectories together |
| **λ ≈ 0** | paths drift in parallel | marginally stable, edge-of-chaos |
| **λ > 0** | paths separate exponentially | chaotic, exponentially sensitive to initial conditions |

The estimator is substrate-agnostic:
```
λ = (1 / τ) · ln( D_final / D_init )
```
where `D_init` is the distance between a point and its nearest neighbour in phase space, and `D_final` is the distance between them τ steps later.

This is one of the very few quantities that gives a **trajectory-level** answer to *"how stable is this dynamical system?"* without needing a closed-form model. Bring trajectories, get λ.

---

## Why λ is interesting for LLM work

Modern LLM evaluation leans hard on **point-wise uncertainty**: perplexity, top-k entropy, log-prob margins, calibration error. These measure spread at a single timestep.

But the failure modes that matter operationally are **trajectory-level**:

- Same prompt, whitespace perturbation → completely different chain of thought
- Multi-step agent, same plan → loop blow-up in one run, success in another
- Self-consistency: 8 sampled CoTs from one prompt, half coherent, half drift
- Long-context reasoning: model is locally confident every step yet globally diverges

These are dynamical-system pathologies, not point-wise ones. Perplexity and entropy can look acceptable while λ reveals a completely different trajectory-level risk profile.

The hypothesis the repo is set up to explore:

> **Compute λ over trajectories generated by an LLM (hidden states, sampled token paths, or chain-of-thought ensembles). Use λ as a structural prompt-risk score that perplexity cannot see.**

### What that buys you

| Use case | Point-wise score | λ adds |
|---|---|---|
| Prompt brittleness audit | "model is uncertain at token t" | "this whole prompt region is exponentially sensitive - any perturbation reroutes the reasoning" |
| Chain-of-thought stability | log-prob of final answer | divergence rate across reasoning steps - drift detector |
| Agent trajectory safety | per-step confidence | does the state trajectory loop, escape, or stabilise? |
| Self-consistency calibration | majority vote | λ of the ensemble = principled stand-in for sampling N → ∞ |
| Eval data curation | hard / easy split by accuracy | hard-and-stable vs hard-and-chaotic - very different items |

λ and entropy are **complementary**, not competing. High entropy + λ ≈ 0 is soft, well-behaved uncertainty. Low or moderate entropy + λ ≫ 0 is the deceptive case: the model can look locally confident while the whole trajectory is fragile.

## The neuro → LLM bridge

The original brain-dynamics use case and the LLM application share the same mathematical skeleton:

| Component | Brain (original) | LLM (bridge) |
|---|---|---|
| State at time t | parcel activations (100-dim) | hidden state, layer activations, or sampled token logits |
| Phase space | 3 gradients + first derivatives → 6D | layer-wise activations or embedding ensemble |
| Trajectory | one fMRI scan | one generation (or many samples from same prompt) |
| Nearby points | similar brain states across time | similar prompts / parallel CoT samples |
| λ > 0 reading | edge-of-chaos cognition | prompt-region instability, hallucination-cascade risk |

The implementation in [`lyapunov_core.py`](lyapunov_core.py) is deliberately substrate-agnostic. Feed it `X ∈ R^(T × d)`, get λ. The neuro pipeline and the LLM demo are two thin wrappers around the same function.

A worked example on synthetic LLM-style trajectories lives in [`examples/llm_bridge_demo.py`](examples/llm_bridge_demo.py).

## Why this matters for *large stochastic data* specifically

For deterministic systems the Lyapunov spectrum is unambiguous. For stochastic systems (which is what we always have: fMRI, market prices, LLM sampling, anything real) the estimator is computing something slightly different - it picks up **finite-time stochastic divergence rate**, sometimes called the *largest local Lyapunov exponent* or *finite-time Lyapunov exponent (FTLE)*.

That is actually the more useful quantity:

- It is well-defined for noisy, finite, irregularly-sampled data.
- It is sensitive to **regime changes** even when global stationarity is broken.
- It separates two kinds of uncertainty that information-theoretic measures conflate: *spread at a moment* (entropy) vs *spread evolving over time* (λ).

In high-dimensional stochastic settings - cortical dynamics, multi-omics time-series, financial regimes, LLM sampling - this is the difference between "the distribution is wide" and "the system is amplifying perturbations". Very different things to operate on.

## Quick start

### Substrate-agnostic API

```python
import numpy as np
from lyapunov_core import largest_lyapunov

# X: numpy array of shape (T, d). Any reconstructed phase space.
X = np.random.randn(1000, 6)
lam = largest_lyapunov(X, tau=3, k_neighbors=5)
print(f"λ = {lam:.4f}")
```

### Original neuro pipeline (Schaefer-100, 3 gradients, HCP)

1. Extract parcel time series (Schaefer parcellation, n=100).
2. Load HCP functional-connectivity gradients (BrainSpace).
3. Project time series into gradient space (Pearson correlation against each gradient).
4. Reconstruct 6D phase space by concatenating correlation and its first derivative.
5. Split data in half, find nearest neighbours in phase space.
6. Track divergence over τ steps.
7. `λ = log(D_final / D_init) / τ`.
8. Group statistics: Shapiro test, t-test, mixed-effect models.

### LLM bridge (synthetic example)

```bash
python examples/llm_bridge_demo.py
```

Generates two synthetic LLM-style hidden-state trajectories - one from a *stable* prompt region, one from a *brittle* region - and shows how λ separates them while their marginal entropy alone under-explains the difference.

## How to read a λ value (cheat sheet)

| You measure | Likely interpretation | What to do |
|---|---|---|
| λ ≈ 0, low entropy | confident and stable | trust the output, light eval |
| λ ≈ 0, high entropy | uniformly uncertain, well-behaved | calibrate; sampling helps |
| λ ≫ 0, low entropy | **deceptive confidence** - small input flips reasoning | red-team; prompt is structurally unsafe |
| λ ≫ 0, high entropy | chaotic exploration | useful for creative tasks, dangerous for reasoning |
| λ < 0 | strong attractor pulling toward one answer | check for mode collapse / sycophancy |
